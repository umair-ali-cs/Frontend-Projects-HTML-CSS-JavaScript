# Chess Game

## Overview
This is a fully functional **Chess Game** developed using **HTML, CSS, and JavaScript**. The game allows two players to play chess on the same device, following standard chess rules including piece movements and captures. The project demonstrates DOM manipulation, event handling, and basic game logic in JavaScript.

## Features
- Interactive chessboard with all standard pieces.
- Player vs Player mode on a single device.
- Highlights possible moves for selected pieces.
- Enforces basic chess rules (legal moves, captures).
- Undo and Reset functionality for moves.
- Responsive and visually appealing UI.

## Usage
- Click on a piece to see its possible moves.
- Click on a valid square to move the piece.

## Technology Stack
- **HTML5** – Structure and layout of the chessboard.
- **CSS3** – Styling the chessboard and pieces.
- **JavaScript (ES6)** – Game logic, piece movement, event handling, and interactivity.
